# Exercises

1. [Jedi Level 1](https://github.com/ryanclove/LearningGoProgramming/tree/master/Exercises/Jedi%20Level%2001) - variables, values, types
    1. Assign values to variables
    2. Declare variables with identifiers 
    3. Scope of variables
    4. Create your own type
    5. Use var keywork to create a variable with an underlying custom type

2. [Jedi Level 2](https://github.com/ryanclove/LearningGoProgramming/tree/master/Exercises/Jedi%20Level%2002) - Programming Fundamentals
    1. Print decimal, binary, and hex
    2. Use operators and expressions
    3. Create typed and untyped constants
    4. Shift bits of an integer variable
    5. Create a string var using a raw string literal
    6. Use ioa to create 4 constants incrementing 4 years
3. [Jedi Level 3](https://github.com/ryanclove/LearningGoProgramming/tree/master/Exercises/Jedi%20Level%2003) - Control Flow (loops, if else, conditions)
    1. For loop to print ints
    2. Print rune code (unicode) 
    3. Create for condition {} loop
    4. Create for {} loop
    5. Print remainders (modulus)
    6. Use else if and else
    7. Use a switch statement
    8. Use a switch expression
    9. Print true/false conditions with &&, ||, !
4. [Jedi Level 4](https://github.com/ryanclove/LearningGoProgramming/tree/master/Exercises/Jedi%20Level%2004) - Grouping Data (slices, maps)
    1. Create an array using a composite literal
    2. Create a slice using a composite literal
    3. Use slicing to create new slices
    4. Append to a slice
    5. Delete from a slice
    6. Use make and append for a slice
    7. Create a slice of a slice to make a multi-dimensional slice
    8. Create a map
    9. Add a record to a map and print using range loop
    10. Delete a record from a map and print using range
5. [Jedi Level 5](https://github.com/ryanclove/LearningGoProgramming/tree/master/Exercises/Jedi%20Level%2005) - Structs
    1. Create own type struct
    2. Store values of a struct in a map
    3. Create new type struct and embed the type in other structs
    4. Use an anonymous struct
6. [Jedi Level 6](https://github.com/ryanclove/LearningGoProgramming/tree/master/Exercises/Jedi%20Level%2006) - Functions
    1. Create func foo and func bar
    2. Create fun with takes in a variadic parameter
    3. Use defer keyword for func
    4. Attach a method to a defined struct
    5. Interfaces & Polymorphism
    6. Anonymous func
    7. Assign a func to a variable
    8. Create a func that returns a func
    9. Use a callback (pass a func into a func as an arg)
    10. Enclose the scope of a variable
7. [Jedi Level 7](https://github.com/ryanclove/LearningGoProgramming/tree/master/Exercises/Jedi%20Level%2007) - Pointers
